```
用户进入页面
    ↓
看到生成器和使用提示
    ↓
需要向下滚动
    ↓
看到快速示例
    ↓
继续向下滚动
    ↓
看到最近生成
    ↓
继续向下滚动
    ↓
看到当前状态
    ↓
❌ 用户体验差，需要多次滚动
```

### 改进后的优势

```
用户进入页面
    ↓
一眼看到所有信息：
  • 左上：使用提示（如何使用）
  • 右上：当前状态（我的配额）
  • 左下：快速示例（快速开始）
  • 右下：最近生成（快速访问）
    ↓
✅ 用户体验优，无需滚动
```

## 💡 设计原理

### 信息层级

```
上行（指导信息）
├─ 使用提示：教用户如何更好地描述需求
└─ 当前状态：告诉用户他们的配额和订阅状态

下行（操作入口）
├─ 快速示例：提供快速开始的模板
└─ 最近生成：快速访问历史表单
```

### 视觉平衡

```
左侧（操作相关）          右侧（信息相关）
├─ 生成器（主操作）       ├─ 使用提示（指导）
├─ 快速示例（快速开始）   └─ 当前状态（信息）
└─ 最近生成（快速访问）
```
