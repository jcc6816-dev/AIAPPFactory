## 🔧 CSS 改动

### 关键改动

```css
/* 改进前 */
.right-card {
    display: flex;
    flex-direction: column;
    gap: 12px;
}

/* 改进后 */
.right-card {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 12px;
}
```

### 卡片高度

```css
.info-card {
    display: flex;
    flex-direction: column;
    min-height: 140px;  /* 确保卡片有最小高度 */
}
```

### 内容间距优化

```css
/* 使用提示和当前状态 */
.info-list {
    gap: 4px;  /* 从 6px 改为 4px */
}

/* 快速示例和最近生成 */
.recent-forms {
    gap: 4px;  /* 从 6px 改为 4px */
}

/* 按钮间距 */
.recent-form-item {
    padding: 6px 8px;  /* 从 8px 10px 改为 6px 8px */
}
```
