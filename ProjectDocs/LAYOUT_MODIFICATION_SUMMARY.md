## 修改内容

### 1. 布局结构变更
**之前**：使用固定的 `xl:grid-cols-[minmax(320px,0.95fr)_minmax(420px,1fr)_minmax(320px,0.92fr)]` 布局
- 中间列使用 `sticky top-24` 定位，容易导致布局抖动
- 列宽分配不均匀

**之后**：使用响应式 `lg:grid-cols-3` 布局
- 三列等宽分布
- 移除了sticky定位，避免布局问题
- 更好的响应式表现

### 2. 三栏结构
```
┌─────────────────────────────────────────────────────────┐
│  左栏（生成器）  │  中栏（预览）  │  右栏（字段配置）  │
├─────────────────┼────────────────┼──────────────────┤
│ • 提示词输入    │ • 单题流预览   │ • 字段详情编辑   │
│ • 主题选择      │ • 实时效果展示 │ • 字段操作按钮   │
│ • 示例提示词    │                │ • 字段属性配置   │
│ • 标题/描述     │                │ • Schema查看     │
│ • 生成/保存按钮 │                │                  │
│ • 字段列表      │                │                  │
└─────────────────┴────────────────┴──────────────────┘
```

### 3. 响应式设计
- **桌面端（1400px+）**：3列等宽布局，所有内容同时可见
- **平板端（1024px-1399px）**：自动调整为2列或单列（由Tailwind的lg断点控制）
- **手机端（<1024px）**：单列堆叠，从上到下依次为生成器、预览、字段配置

### 4. 代码改进
- 移除了 `sticky top-24` 定位，改为正常流布局
- 添加了清晰的注释标记三栏区域
- 保持了所有功能逻辑不变
- 无API或状态管理改动
