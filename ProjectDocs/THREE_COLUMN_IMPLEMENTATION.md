## 🔧 技术实现

### CSS 关键代码

```css
/* 左中右三栏布局 */
.stage-1-content {
    display: grid;
    grid-template-columns: 1fr 1.5fr 1fr;
    gap: 20px;
    flex: 1;
    min-height: 0;
}

/* 生成器卡片突出 */
.generator-card {
    background: linear-gradient(135deg, rgba(238, 242, 255, 0.5), rgba(255, 255, 255, 1));
    border: 2px solid #3b82f6;
    box-shadow: 0 8px 24px rgba(59, 130, 246, 0.15);
}

/* 响应式 - 平板版 */
@media (max-width: 1200px) {
    .stage-1-content {
        grid-template-columns: 1fr;
    }
    
    .stage-1-content .column:nth-child(2) {
        order: -1;  /* 生成器移到最上面 */
    }
}

/* 响应式 - 手机版 */
@media (max-width: 768px) {
    .stage-1-content {
        grid-template-columns: 1fr;
    }
}
```
